HTML is an acronym which stands for Hyper Text Markup Language which is used for creating web pages and web applications


CSS stands for Cascading Style Sheets. It is a style sheet language
A CSS class is an attribute used to define a group of HTML elementsin order to apply unique styling and formatting to those elements with CSS.
In CSS, inheritance controls what happens when no value is specified for a property on an element
CSS properties can be categorized in two types: inherited properties, which by default are set to the computed value of the parent element.
The inherit keyword specifies that a property should inherit its value from its parent element.
